Domain Name Generator

This is a domain name generator which at the press of the button, will list different domain names based on set pronouns, adjectives and nouns.

Post-fixed to these names will be a top-level-domain such as .com, .org etc.

You can optionally add a domain-hack which will include a selected number of characters on the domain name as a sub-top-level-domain.
